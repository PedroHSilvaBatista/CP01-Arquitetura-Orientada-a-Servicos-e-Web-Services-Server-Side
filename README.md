# CP01 - Arquitetura Orientada a Serviços e Web Services (Server Side)

## 📌 Descrição
Este projeto representa o **lado do servidor** de uma aplicação baseada em **Arquitetura Orientada a Serviços (SOA)** utilizando **Web Services SOAP** em Java.

O sistema disponibiliza serviços relacionados a uma aplicação fictícia de gerenciamento de vinhos chamada **Winery**.

Os serviços são publicados localmente utilizando **JAX-WS (Jakarta XML Web Services)**.

## 🧰 Tecnologias Utilizadas

- Java
- Maven
- JAX-WS (Jakarta XML Web Services)
- SOAP
- IntelliJ IDEA
